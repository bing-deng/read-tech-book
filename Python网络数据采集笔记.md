##### 网络连接
* ``浏览器层``  `` 网络连接层``
* 浏览器是一个比较年轻的发明，始于1900年的Nexus浏览器
* 网络浏览器：创建信息的数据包，发送他们，然后把获取的数据解释成漂亮的图像、声音、视频、文字。
* 网络浏览器就是代码：而代码是可以分解的，可以分解成许多基本组件，可重写，重用，以及做成我们想要的东西

* 最简单的爬虫代码

```
from urllib.request import urlopen

html = urlopen('https://google.com')
print(html.read())

```
